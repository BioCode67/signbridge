# 넷 챌린지 캠프 시즌13 — 기획안 초안 (SignBridge)

> HWP 양식(참가신청서 및 기획안)에 옮겨 적기 위한 **제출용 콘텐츠 초안**입니다.
> 접수: ~6/22, 제출물 = 참가신청서 + 기획안. (데모 웹앱은 11/18 최종 현장시연용 실증물)

---

## 1. 과제명
**SignBridge — KOREN 저지연망 기반 재난정보 실시간 AI 수어(KSL) 통역·송출 서비스**

## 2. 추진 배경 및 필요성
- 국내 **등록 청각장애인 약 42만 명**(보건복지부). 다수 농인에게 **한국수어가 모어(母語)**이며, 2016년 **한국수어법**으로 국어와 동등한 공용어로 인정됨.
- 그러나 재난문자(CBS)·재난방송은 **텍스트·음성 중심**이라 농인에게 즉시 도달하지 못함. 한국어 문해력 편차로 텍스트만으로는 정보 격차가 큼.
- 사람 수어통역사는 **심야·지역·동시다발 재난**에서 24시간 전 채널 실시간 대응이 불가능 → **골든타임 공백**.
- → 재난정보를 **즉시 한국수어 아바타로 변환해 전국에 실시간 송출**하는 자동화 인프라가 필요.

## 3. 아이디어 개요
재난 텍스트(재난문자/방송 자막) → **AI가 한국수어 글로스·동작으로 변환** → **3D/2D 수어 아바타 생성** → **KOREN 저지연 백본으로 전국 NIA POP 10개소에 동시 송출** → 방송·모바일 앱·전광판·키오스크 다채널 표출.

## 4. KOREN 활용 방안 (핵심)
| KOREN 자원 | 활용 |
|---|---|
| **AI Network Lab(판교) HPC·GPU** | 수어 변환 AI 추론 + 3D 아바타 실시간 렌더링(GPU) |
| **KOREN 저지연 백본** | 변환 결과를 **종단 0.5초 목표**로 전국 다지점 실시간 멀티캐스트 |
| **NIA POP 10개소(서울·대전·대구·부산·광주 등)** | 지역 엣지 송출점 → 지역별 동시 표출, RTT 최소화 |
| **대용량·고품질 회선** | 다채널 수어영상(아바타) 동시 스트리밍 |
- 일반 상용망 대비 **저지연·고대역·다지점**이 본질 → "네트워크가 곧 서비스 품질"인 구조로 KOREN 적합.

## 5. 기술 구성
- **데이터**: AI Hub「재난 안전 정보 전달을 위한 수어영상」 — 실제 농인 수어의 OpenPose 키포인트(2D/3D) + 형태소·비수지(표정) 라벨. 본 데모는 지진·태풍·호우·대설·산불·미세먼지 등 **17개 재난 문장** 수록.
- **변환/렌더**: 키포인트 → 리깅 3D 휴머노이드(VRM) 본 회전 리타게팅(상반신·손가락·표정) + 2D 캔버스 아바타(비교용).
- **프런트엔드 실증**: Vite+React+TypeScript+Three.js. 히어로 네트워크 시각화, 수어 데모(아바타 11종 선택·2D/스켈레톤/3D), 작동원리·시스템 구성도, 실시간 네트워크 텔레메트리.
- **시스템 흐름**: 재난입력 → KOREN 저지연망 → HPC·GPU AI변환 → 수어 아바타 → 전국 다채널 송출.

## 6. 차별성
- **실데이터 기반**(AI Hub) — 가상의 모션이 아닌 실제 농인 수어 키포인트.
- **네트워크 중심 설계** — 단일 서버 데모가 아니라 KOREN 저지연 다지점 송출을 전제한 구조.
- **다채널·다지역 동시성** — 방송/앱/전광판/키오스크, 전국 POP 동시 표출.
- **확장형 데이터 파이프라인** — 문장 추가가 코드 수정 없이 가능(manifest 기반).

## 7. 상용화·기대효과
- **공익(B2G)**: 행안부·지자체 재난방송, 재난문자 연계 표준 수어 송출.
- **확장**: 공공안내방송·교통·의료 등 실시간 수어 통역 일반화.
- **사회적 가치**: 농인의 재난 안전 정보 접근권 보장(정보 접근성·SDG·디지털 포용).

## 8. 추진 일정(안)
- 7~8월: KOREN 회선·HPC 연동, 실시간 송출 파이프라인 PoC(POP 다지점)
- 8월: 중간평가 / 챌린저데이 — 저지연 실증
- 9~10월: 변환 정확도·아바타 품질 고도화, 다채널 표출
- 10/30: 현장 시연(재난문자 입력→전국 POP 동시 수어 송출)
